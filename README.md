## Setup

1. Go to [GitHub Settings > Pages](https://github.com/Cmarala/TWAVoterApp/settings/pages)
2. Set Source to "GitHub Actions"
3. The app will be available at: https://Cmarala.github.io/TWAVoterApp/

## Telegram Bot Setup

1. Open [@BotFather](https://t.me/BotFather) on Telegram
2. Create a new bot or use existing one
3. Use `/setmenubutton` command
4. Set the Web App URL to: `https://Cmarala.github.io/TWAVoterApp/`

## Development

This is a simple HTML/CSS/JavaScript Telegram Web App that follows Telegram design principles.
